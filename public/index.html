本地控制界面

一个界面覆盖所有 Agent Reach 路由。

搜索、社交平台、GitHub、网页读取、视频字幕、招聘检索、环境诊断和更新检查，全部通过同一层可审计命令模板执行。

打开运行器
生成的命令就绪
make doctor
搜索社交视频开发

能力覆盖

从 skill reference 映射到图形界面。

每个能力卡片都绑定固定命令模板，不执行任意 shell 文本。

先选路由访问需要登录态的平台前，会先判断当前可用后端。
面向读取主要用于调研、检索、转录和诊断，不用于账号写操作。
多后端OpenCLI、MCP、公开 API 和平台 CLI 都作为独立可见路由暴露。

运行器

选择一个能力。

选择上方能力卡片，填写必要字段，然后在本地运行或复制命令。

输出
尚未运行命令。

隐藏 CLI 入口

按 Ctrl+K 打开命令面板。

命令面板提供与 Makefile 兼容的终端命令。默认隐藏，需要时再打开。

make search QUERY="agent reach"make youtube URL="https://youtube.com/watch?v=VIDEO_ID"make package
